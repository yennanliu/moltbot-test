# Simple Todo List App

This is a minimal **HTML/JavaScript** todo list application.

## Usage

1. Clone the repo or open the site URL below.
2. Add todos using the input field and **Add** button (or press Enter).
3. Click a todo item to mark it completed (toggle).
4. Double-click a todo item to remove it.
